# Togetli Product Context

## What Togetli is

Togetli is a mobile app for group trip planning.

It helps groups:

* make decisions (dates, budget)
* coordinate logistics (travel, itinerary, info)

It works alongside group chat — not replacing it.

---

## Core problem

Group trip planning is:

* chaotic (group chats)
* fragmented (multiple tools)
* unclear (no single source of truth)
* repetitive (same questions asked repeatedly)

---

## Core solution

Togetli provides:

### Phase 1 — Decision

* collect availability + budgets
* structure group input
* finalize plan

### Phase 2 — Coordination

* travel details (who arrives when)
* itinerary (live, editable)
* shared info (links, notes)

---

## What Togetli is NOT

* not a messaging app
* not a Canva replacement
* not a full booking platform

---

## Target users

* creator (organizer)
* members (participants)

(planner role simplified for MVP)

---

## Product goals (MVP)

* make group decisions clear
* reduce chat chaos
* provide one place for coordination
* create repeat usage AFTER planning

---

## Product principles

* clarity over complexity
* structure over chaos
* coexist with existing tools
* lightweight but powerful

---

## Current product scope

Focus:

* trip creation
* poll system
* coordination layer (NEW)

Not built:

* destination system
* advanced planning tools
* integrations

---

## Key insight

The problem is not communication.

It is:

→ lack of structure and coordination in groups

Togetli solves that.
